# E-Commerce-Performance-dashboard

## 🚀 Project Overview
Built an end-to-end ETL and analytics solution to analyze e-commerce sales data using Python, PostgreSQL, and Power BI.

---

## 🛠️ Tech Stack
- Python (Pandas)
- PostgreSQL
- Power BI
- DAX

---

## 🔄 ETL Pipeline
- Extracted raw CSV data
- Cleaned and transformed using Pandas
- Validated revenue calculations
- Loaded into PostgreSQL database

---

## 📊 Dashboard Features
- Total Revenue, Orders, AOV KPIs
- Month-over-Month (MoM) Growth
- Latest Month Revenue tracking
- Revenue by Category, City, Payment Method
- Top Customers analysis

---

## 📈 Key Insights
- Revenue grew 9% MoM indicating stable growth
- Delhi is top performing city
- Net Banking & UPI dominate payments
- Balanced category contribution
